## Directives

Extra HTML tags and attributes Angular provides, and we have Two types:

## Built-in Directives

- **Component Directives**
- **Structural Directives**
  - `ngIf`
  - `ngFor`
  - `ngSwitch`
- **Attribute Directives**
  - `ngClass`
  - `ngStyle`
  - `ngModel`

## Variables for \*ngFor

1. `Count`
2. `index`
3. `trackBy` (takes a function that returns a unique value)
